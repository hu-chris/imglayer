freeCodeCamp Image Search Abstration Layer Project

User Stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example Image Search Usage:

  • https://imglayer.glitch.me/search/lolcats%20funny?offset=7
  • https://imglayer.glitch.me/search/polar%20bear/

Example Image Search Output:

  • { "imgUrl": "https://www.google.com/maps/about/images/treks/churchill3-carousel.jpg", "altText": "Discover the world with Google Maps. Experience Street View, 3D Mapping, turn-\nby-turn directions, indoor maps and more across your devices.", "pageUrl": "https://www.google.com/maps/about/behind-the-scenes/streetview/treks/churchill-canada/" }

Example History Usage:

  • https://imglayer.glitch.me/history

Example History Output:

  • {{ "searchterm": "polar bear", "datetime": "2018-01-10T08:21:47.827Z" }
